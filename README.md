# Loan Default Risk Analysis Project

I built this project to connect my finance experience with data skills.  
I use real loan application data.  
I show risk patterns with visuals.  

## Project Goal  
I compare income and credit amount.  
I highlight default customers.  
This helps banks reduce bad loans.  

## Data Source  
I download the dataset from Kaggle.  
The name is Home Credit Default Risk.  
It contains over 300,000 loan records.  

## Tech Stack  
I use Tableau Public for visualization.  
I use scatter plots with color by default flag.  
I use box plots for income distribution.  

## Steps I Followed  
I connect the CSV file in Tableau.  
I clean missing values to 0.  
I create a scatter plot for credit vs income.  
I create a box plot for income by default status.  
I combine charts into one dashboard.

## Key Findings  
Higher credit amounts link to more defaults.  
Default customers have lower average income.  
Banks can use these patterns to check risk faster.
